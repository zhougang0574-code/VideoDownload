YouTube 视频工具

上传功能需要先在 Google Cloud Console 配置一次 OAuth 凭据（下载功能不需要这一步）：
1. 打开 console.cloud.google.com，新建或选一个项目
2. "API和服务 → 库"，启用 YouTube Data API v3
3. "API和服务 → OAuth 同意屏幕"，用户类型选"外部"，发布状态留"测试中"，把自己的账号加进"测试用户"
4. "API和服务 → 凭据 → 创建凭据 → OAuth客户端ID"，应用类型必须选"电视和有限输入设备"
5. 把拿到的 Client ID 和 Client Secret 填在下面，保存一次即可

用手机扫码打开确认页面，然后手动输入上面这个验证码确认登录
